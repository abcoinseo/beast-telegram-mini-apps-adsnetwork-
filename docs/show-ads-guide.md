# How to Show Ads

1. Load Ads from your app dashboard.  
2. Call the SDK function (see [SDK Integration](sdk-integration.md)).  
3. Reward users after ad completion.  
4. Test ads using **Test APP ID** from dashboard.  
5. Use multiple placements for maximum earnings.

**Best Practices:**

- Place ads where users are likely to complete the view.  
- Avoid overloading with too many ads.  
- Encourage full completion for reward tracking.
